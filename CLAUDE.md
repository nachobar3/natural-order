# Proyecto: Página web completa - Enero 2026

## Stack preferido
- HTML5 + CSS3 + JavaScript vanilla (o con framework si lo pido)
- Tailwind CSS (via CDN o instalación)
- Diseño responsive first
- Accesibilidad WCAG nivel AA
- Optimización básica performance

## Filosofía
- Mobile-first
- Código limpio y comentado
- Usar componentes reutilizables
- Pedir confirmación antes de crear más de 3 archivos o hacer cambios grandes

## Comandos útiles del proyecto
npm run dev     → levantar servidor local (si usamos Vite/React/etc)
npm run build   → generar versión producción
